Scalaz日本語ドキュメント

\/

  • ソースコード
  • scaladoc
  • 記号のクラス名でとっつきにくそうですが、かなり多く使われている、とても便利なクラスです
  • 読み方はDisjunctionあるいはEitherです
  • 数学の論理和 ∨ の記号由来です
  • \/ 自体はsealed abstract classであり、\/-と-\/というサブクラスが存在する
  • 右に-がくっついているのがRight、左に-がくっついているのがLeft
scala> import scalaz._
import scalaz._

scala> val a: Int \/ String = \/-("foo")  // rightを生成
a: scalaz.\/[Int,String] = \/-(foo)

scala> val b: \/[Int, String] = a // 型は中置記法で書かれることが多いが、中置記法を使わなくても書ける
b: scalaz.\/[Int,String] = \/-(foo)

scala> val c: Int \/ String  = \/.right("foo") // rightというメソッドもある
c: scalaz.\/[Int,String] = \/-(foo)

scala> val d = a.map(_ + "bar") // rightをmapすると反映される
d: scalaz.\/[Int,String] = \/-(foobar)

scala> val e: Int \/ String  = \/.left(42)
e: scalaz.\/[Int,String] = -\/(42)

scala> e.map(_ + "bar") // leftをmapしても変わらない
res0: scalaz.\/[Int,String] = -\/(42)

scala> e.leftMap(_ * 100) // leftの場合にmapするには、leftMapを使う
res1: scalaz.\/[Int,String] = -\/(4200)
